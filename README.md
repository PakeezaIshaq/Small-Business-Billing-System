# Small Business Billing System

This project is a simple yet functional Billing System implemented using Object-Oriented Programming principles in C++. It allows small business owners to input various services, their unit rates, and quantities to generate a detailed receipt with a grand total.

### Features
- Add multiple services with name, unit rate, and quantity.
- Calculate total for each service.
- Generate a comprehensive receipt displaying all services and the total bill.
- Input validation for unit rates and quantities.

### Technologies Used
- C++ (Standard Library)
